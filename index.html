LinkedIn Helper

A tool to find things on LinkedIn quickly and with less efforts

by Shishir Khandelwal


A B O U T

The tool goes through your linkedIn connections list, connection posts and lets you apply filters

to see only the posts that are relevant to you. You can like/comment on indivisual posts

or on multiple posts with a click of one button. In this way you can save time and efforts

to find posts/profiles relevant to you.

It uses Bs4, Selenium and chromedriver for automation, PyQt5 for UI and is written on Python


It's free and requires no installation, just follow the instruction given below to get started


GET STARTED

1. Download the file from Link

2. Extract file

3. Check the version of the Chrome.

To check Chrome version

Open Chrome - Type "chrome://version/" inside the search bar. Hit Enter.

Note version from the first line, it would be one of 80,81 or 83

4. Inside the extracted contents, open 'WinPython Command Prompt'

5. If chrome version was 80 - Type "cd win_64_chrome_80

If chrome version was 81 - Type "cd win_64_chrome_81

If chrome version was 83 - Type "cd win_64_chrome_83

6. Hit Enter.

7. Type "python app.py". Hit Enter.


The application should start now. If there is any issue, write to me at shishir29121997@gmail.com
